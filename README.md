# choco-app

Библиотека для взаимодействия веб-приложений с Choco SuperApp. Работает только в браузерном окружении.

## Установка

```console
npm install choco-app
```

### `clearCache`

Type: `void`<br>

Очищает кэш приложения.

```typescript
import { clearCache } from 'choco-app';

clearCache();
```
